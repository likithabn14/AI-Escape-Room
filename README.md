# 🧩 AI Escape Room

AI Escape Room is an interactive puzzle-solving game built with **Streamlit**. Players race against a countdown timer to solve different types of puzzles, earn points, use hints wisely, and escape before time runs out. Every new game generates a different set of puzzles, making the experience fun and replayable.

---
DEMO LINK: https://ai-escape-room-m4cfx7g4eiyhrrx4f5jxkj.streamlit.app/

## 🚀 Features

- 🎮 Interactive Escape Room gameplay
- ⏳ 5-minute countdown timer
- ❤️ 5 Lives system
- 💡 3 hints for every puzzle
- ⭐ Score tracking
- 🧩 Random riddles
- ➕ Random math puzzles
- 🔐 Caesar cipher decoding
- 😀 Emoji puzzles
- 🧠 Logic puzzles
- 🔄 Random questions every time you play
- 🎉 Confetti celebration on winning
- 🏆 Leaderboard support
- 🌙 Cyber-themed dark UI

## 🎯 How to Play

1. Enter your name and start the game.
2. Solve each puzzle before the timer ends.
3. Use hints carefully whenever you get stuck.
4. Earn points for every correct answer.
5. Avoid losing all your lives.
6. Complete all puzzles to escape successfully.
7. Click **Play Again** to receive a new random set of puzzles.
